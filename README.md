# Titanic-EDA
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand passenger demographics, handle missing data, and identify patterns affecting survival.
The analysis includes data cleaning, statistical exploration, visualizations, and automated data profiling using YData Profiling.
Objectives:
Understand the structure and distribution of the dataset
Handle missing values using appropriate statistical methods
Identify patterns and relationships between features
Generate an automated profiling report for deeper insights
Tools & Technologies:
Python
Pandas
NumPy
Matplotlib
Seaborn
YData Profiling
Google Colab
Conclusion:
This project demonstrates a complete EDA workflow and builds a strong foundation for applying machine learning models on real-world datasets.
